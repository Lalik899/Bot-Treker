# Bot-Treker
Добавить привычку: создать новую привычку, бот попросит ввести название.
Мои привычки: показывает список всех добавленных привычек.
Отметить выполнение: позволяет отметить, что привычка выполнена сегодня.
Удалить привычку: удаляет выбранную привычку из списка.
Статистика: показывает, сколько дней выполнена каждая привычка.
Напоминания: устанавливает время, когда бот будет присылать напоминание.
Профиль: показывает имя пользователя.
Статистика бота (админ): показывает количество пользователей.
Рассылка (админ): отправляет сообщение всем пользователям.
Выйти (админ): возвращает в обычное меню.
