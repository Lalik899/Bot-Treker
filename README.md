# Bot-Treker
Кнопки:
  Добавить привычку:
  Пользователь вводит название новой привычки.
  Привычка добавляется в список, появляется в статистике и для отметки выполнения.

  Мои привычки:
  Показывает текущий список всех привычек пользователя.
  Просто «список», без действий.

  Отметить выполнение:
  Inline-кнопки со списком привычек.
  Нажатие на привычку отмечает её выполнение на сегодня.
  Каждое нажатие сохраняется в статистике.

  Удалить привычку:
  Inline-кнопки со списком привычек.
  Позволяет удалить выбранную привычку вместе с её логами.

  Статистика:
  Показывает, сколько дней была выполнена каждая привычка.
  Считает уникальные дни из логов выполнения.

  Напоминания:
  Пользователь вводит время (HH:MM).
  Бот присылает сообщение в это время: «Пора отметить привычки!».
  Работает как фоновый таймер.

  Профиль:
  Показывает имя пользователя.
